## 技术特点

1. **纯前端实现**: 无需服务器，使用LocalStorage存储数据
2. **与手机端风格一致**: 相同的配色方案和设计语言
3. **模块化设计**: 清晰的文件结构和职责划分
4. **响应式布局**: 适配不同屏幕尺寸
5. **模拟数据**: 完整的测试数据，方便开发调试

## 使用说明

### 启动方式
1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000

   # 使用Node.js
   npx http-server

   # 使用VS Code Live Server插件
   ```

### 登录测试
1. 打开 `index.html`
2. 点击测试账号区域自动填充
3. 或手动输入：
   - 手机号: 13800000001
   - 密码: 123456
4. 点击登录

### 数据说明
- 所有数据存储在浏览器LocalStorage中
- 清除浏览器数据会重置所有信息
- 模拟数据会在首次加载时自动初始化
